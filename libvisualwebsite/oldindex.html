About Libvisual
Libvisual is an abstraction library that comes between applications and audio visualisation plugins.

Often when it comes to audio visualisation plugins or programs that create visuals, they depend on a player or something else; basically, there is no general framework that enabled application developers to easily access cool audio visualisation plugins. Libvisual wants to change this by providing an interface towards plugins and applications; through this easy to use interface applications can easily access plugins and, since the drawing is done by the application, it also enables the developer to draw the visual anywhere he wants.

The framework also allows you to morph to different plugins and mix two at once; all kinds of neat tricks are possible using this method.
Latest News
libvisual-gforce available
20 September 2004
We've released libvisual-gforce. The libvisual port of the well known G-Force visualisation plugin. It's variety is awesome, achieved through presets. This plugin is a must have for sure! In order to get it installed, ofcourse install libvisual 0.1.6 and you probably want libvisual-xmms 0.1.6 as well! Find them at the files section.
libvisual 0.1.6 released
10 September 2004
It has been a long time, and lots have happened! But you can already guess. We've finally released libvisual 0.1.6, libvisual-plugins 0.1.6, libvisual-nebulus 0.1.3 and libvisual-xmms 0.1.6. You can find it at the files section at the project page! Don't forget to remove the old libvisual plugins first since this release is not compatible at all. Do this by removing %prefix/lib/libvisual/* (where prefix is probably something like /usr or /usr/local).

What also might be worth noticing is that Amarok is using us as their visualisation framework! for more info about amarok here.

When you download libvisual, don't forget to download libvisual-plugins as well because that is where the art is! (visuals that is visuals). To use this within xmms download the libvisual-xmms plugin!

Another cool thing is that Salsaman of LiVES fame is working on libvisual/livido intergration! This means that libvisual will soon be useable in video editing and live VJing!

But that is not all! We updated the API documentation as well!
